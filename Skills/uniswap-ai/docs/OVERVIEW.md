# Uniswap AI Overview

Source reference: https://github.com/Uniswap/uniswap-ai

## Summary
Uniswap-specific AI tools for developers and AI agents integrating the trading API.

## Key plugin areas
- uniswap-hooks
- uniswap-trading
- uniswap-cca
- uniswap-driver
- uniswap-viem

## Install examples
- `npx skills add Uniswap/uniswap-ai`
- `plugin marketplace add uniswap/uniswap-ai`
