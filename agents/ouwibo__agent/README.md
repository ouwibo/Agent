# Ouwibo Agent

Ouwibo Agent is a public-facing AI agent website built for a polished, international brand.

## What it does
- Presents the Ouwibo Agent brand clearly
- Provides a public chat surface
- Supports manual model selection and automatic rotation
- Runs as a Cloudflare-ready web worker

## Key capabilities
- Brand-first landing page
- Chat API at `/api/chat`
- Health check at `/health`
- Qwen/DashScope-compatible backend routing

## Example usage
- Visit the homepage to learn what Ouwibo Agent is
- Call `/api/chat` from your app
- Choose a model manually or let the backend rotate automatically

## Notes
This submission is designed for a public, professional web presence and keeps vendor details behind the scenes.
